# 📊 E-Commerce Sales Analysis Dashboard (Power BI + SQL)

## 🔍 Project Overview

This project presents an end-to-end **E-Commerce Sales Analysis** using **SQL** and **Power BI**.
The goal of this project is to extract meaningful insights from raw transactional data and transform them into an interactive dashboard that supports **data-driven business decisions**.

This portfolio demonstrates skills in:

* Data extraction and querying (SQL)
* Data transformation and cleaning
* Data visualization (Power BI)
* Business insight generation

---

## 📁 Files Included

This project consists of 3 main components:

1. **SQL File**

   * Contains all queries used for:

     * Data cleaning
     * Data transformation
     * Aggregations and analysis

2. **Power BI Dashboard (PDF)**

   * Visual representation of:

     * Sales performance
     * Profit trends
     * Customer behavior
     * Product analysis
     * Regional performance

3. **Business Insights PDF**

   * Key insights derived from the dashboard
   * Actionable recommendations for business growth

---

## 📊 Dashboard Highlights

### 🔹 Overall Performance

* **Total Sales:** 2M
* **Total Orders:** 10K
* **Total Profit:** 521K
* **Profit Margin:** 25.9%
* **Average Order Value:** 122.58

---

### 📅 Sales & Profit Trends

* Strong growth observed in **later months (Oct–Dec)**
* Seasonal trend indicates **peak sales in Q4**
* Profit remains relatively stable with slight fluctuations

---

### 🛍️ Product Analysis

* Identified:

  * **Top-performing products** driving revenue
  * **Low-performing products** with minimal sales
  * Products with **high profit margins (>50%)**
* Helps in:

  * Inventory optimization
  * Product strategy decisions

---

### 👥 Customer Insights

* Top customers contribute significantly to revenue
* Some customers generate **low profit despite sales**
* Opportunity to:

  * Improve customer targeting
  * Increase retention strategies

---

### 🌍 Regional Analysis

* Regions like **South, Central, and North Asia** perform well
* Some regions show:

  * Low sales
  * Low profitability
* Useful for:

  * Market expansion decisions
  * Regional strategy optimization

---

### 📈 Yearly & Monthly Trends

* Sales peaked in **2012**
* Slight decline observed in later years
* Profit margin remains consistent (~25–27%)
